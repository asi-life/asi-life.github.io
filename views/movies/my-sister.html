【电影】 我的姐姐观后感

差不多的贝壳 2021-04-19 我的姐姐张子枫

4月2号一个还算特殊的日子看了妹妹独挑大梁演的电影——我的姐姐。

# 电影主题

我的姐姐，主要围绕重男轻女，以及女性的梦想和家庭为主生活的两难选择问题为核心的展开。

# 人物介绍

  • 安然(妹妹饰演)：一个刚读完大学不久，打算考研去北京开始自己新生活的非常独立要强的女生。

  • 安子恒(安然的弟弟)：在姐姐大学的时候出生，在父母意外车祸出事前未和姐姐见过一次面的幼儿园孩子。性格调皮，后期又很聪明的小孩。

  • 姑妈(安然爸爸的姐姐)：作为一个姐姐，为了家里的弟弟，放弃了自己学业，生活。一个为了整个家庭付出了相当多的老实妇女。

  • 舅舅：渣男，非常喜欢钱，打牌。跟很多现实渣男一样，对待别人很渣，对待家里人还算上心，但是也很坑的一个人。

# 整体观感

       前半段的剧情一直在制造冲突，安然与亲戚的冲突，安然与弟弟的冲突，安然与工作同事的冲突，安然与男朋友以及他家庭的冲突，以及安然梦想与要照顾弟弟的冲突。此段看的过程感觉上有点难受，强行通过剧情冲突来把观众的情绪提上来，某些行为我觉得不是安然这个性格很独立的角色会做出来的事情。

       中间剧情为就是通过安然和弟弟的相处，和一些生活，工作小细节来降低安然和各个点之间的矛盾。此段的整体感觉是缺少了一个真正过渡期。

       后半段主要讲述为安然与其他角色之间的相互理解，以及安然最终在梦想，帮弟弟找领养家庭和照顾弟弟之间的选择。这部分整体观感还行。特别是能带入安然角色，有弟弟妹妹的家庭。

       整部剧各个角色演技都还行，塑造最好的两个人物为姑妈和舅舅。但是剧情上不是特别好，只能说中规中矩，大部分观众都能猜到后续的发展。建立冲突的时间太长，导致没有一个好的过渡期来缓解冲突，以及部分剧情略显突兀，如一个幼儿园的被惯坏的弟弟给姐姐泡姜茶，以及姐姐在医院发生的事情。对于重男轻女，以及女性的梦想和家庭为主生活的两难选择两个问题上能让人深思。

# 剧情介绍

       开头一场车祸，妹妹的父母丧生在了车祸中。以此来展开姐姐和弟弟的相遇，姐姐在自己梦想和照顾弟弟的选择之间开启了一系列的故事。

       安然的家庭是一个重男轻女的家庭，姑妈放弃梦想照顾弟弟(安然的爸爸)。安然的父母希望生个男孩，但是二胎政策没放宽的情况下，父母就逼迫安然装瘸子(身体残疾，然后被送养)，安然不配合的情况下多次被打，甚至带去学习游泳想淹死安然，以及弟弟要出生时，想要姐姐照顾弟弟，将大学志愿改了，被迫留在了四川。安然在这样的环境下变得很独立，靠自己读完了大学四年。

       剧情开始，安然并不想像姑妈一样为了弟弟放弃了自己想法，而是想找家庭去领养弟弟。并且开始与弟弟相处的过程也并不愉快，中间甚至是坚定了去送养。但是后续在逐渐相处的过程中，弟弟慢慢依赖姐姐后，以及弟弟的泡姜茶后也慢慢舍不得去送养(这一段感觉过渡的并不是特别完美，而且不是很合理)。后续弟弟为了姐姐去北京追逐自己的梦想，答应了领养，并故意与姐姐闹翻。但是最终在安然即去北京去见弟弟，领养家庭在要求姐姐不能再与弟弟见面的条件上签订协议时，姐姐最后带弟弟跑出来。虽然剧情没有显示最终的选择。但更偏向于姐姐选择照顾弟弟。

# 感情线

       安然有一个家庭幸福且爱她，包容她的男朋友，但同时对于安然不够了解，且其家庭更希望两人早点结婚生子。男朋友不敢将两人打算考研去北京的事告诉家里人，没有太多主见。导致最终两人分手。且两人有吻戏(插刀=.=)

# 结合自己来看

       以前一直希望自己有一个亲姐，然后一直跟在屁股后面，学着她去认识一切东西，也希望能和她去说一些自己脑袋奇怪的想法，但是并没有。

       从小就还算乖，后面跟着同学上网，开始了漫长的网瘾时间，偷偷摸摸，也导致后续干了很多的傻事。可能因为干了太多傻事，在我初中的时候，迎来了一个小我十三岁半的妹妹。由于初，高中一直是留校，每半个月甚至一个月回来一次。所以与妹妹的相处时间并不长。但是又很奇特，可能是血缘关系的原因，妹妹特别粘我，而且我并没有当好一个哥哥，但是感觉很神奇。所以可能对于剧中安然和弟弟的冲突感觉有点难受吧，对于安然后续不舍得弟弟的桥段虽然感觉过渡不完美，但是很可能就是那奇怪的血缘关系在吧。所以对于剧中，弟弟和姐姐的那些好的片段还是观感不错的。

       最有感触的其实最后姐姐的抉择了。我也有想过安然那种逃离一个地方，开始自己的生活，玩玩全全自己一个人过的想法，完全不去关心其他事情，自己无论什么结局都无所谓的想法。不过在代入安然这个角色的时候，没有了父母，明知道自己还有一个亲弟弟，却再也不能见到，然后一个人无依无靠的开始新的生活时，发现真的很难很难选择丢下，哪怕有过不愉快，哪怕相处的时间很短。所以希望最后的结局是姐姐带着弟弟一起去北京生活，追寻自己的梦想，虽然会累一万倍。

# 影评

       这是完全以妹妹为核心演绎的电影，有闪光点，很多地方的情绪很不错，但也有不足，妹妹的年龄来看还不太适合演姐姐。同时因为妹妹并不是四川人，学习说方言并没有老演员那么拿捏的准，所以妹妹在说方言的时候会有部分感觉和舅舅(肖央)，姑妈(朱媛媛)那么舒服。妹妹演技还是值得肯定的~~其他演员演技也都在线，但是剧情上感觉一般，很多地方都能猜到情绪会如何发展，特别是开头强制性的冲突展开后。算是中规中矩吧。
